# Swirrl Auth0

See [Auth0 docs](https://github.com/Swirrl/swirrl-auth0/blob/master/doc/auth0.org)

# Usage

Add a dep like:

```
com.swirrl/auth0 {:git/url "git@github.com:Swirrl/swirrl-auth0"
                  :git/sha "2c8147a23b5088efda72cf7027ea4d830c1237e8"
                  :git/tag "v1.0.1"}
```
